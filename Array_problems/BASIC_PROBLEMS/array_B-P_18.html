Checking if a particular element exist in Array.

let MyArray = ["N","A","M","E"];
let arrayHas = MyArray.includes("M");
console.log(arrayHas);

RESULT :- True

INSIGHT:

To know if certain element exist in an array or not we use a property of an array that is 'Includes()'. What this property do is go through all the elements of the array one by one and when it finds the perfect match "===" through comparison it stops and return 'true' else 'false'.
