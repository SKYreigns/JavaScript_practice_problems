Acessing The Last Element Of myArray.

For accessing the last element of myArray, we we don't know the exact lenght of array.

We can achieve that by doing this.


let myArray = ["", "s","o","m","e","t","h","i","n","g"];
console.log(myArray[myArray.length - 1]);

"myArray.lenght": gives the total lenght of the array. So,to get the last one we need to subtract 1 from the total lenght . Counting starts from index 0 and goes to 'n' so the length will be "n+1".{counting =>> 0 = 1 ; 1-n = n => total counting = n+1}
