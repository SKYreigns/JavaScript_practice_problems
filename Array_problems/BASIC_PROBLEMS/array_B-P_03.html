Accessing The First Element Of Array

We can access any element of array with some element is by writing the index of that particular element. The index of the Array starts with ZERO {0} and goes to total lenght of array minus 1.

So,to get the first element of the array we need to write the index 0.

Here's how you can achieve it.

let myArray = ["first", "second","third"];
console.log(myArray[0]);
