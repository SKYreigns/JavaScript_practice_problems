Iterating Each Element Using fOR Loop.

using "FOR" loop we can easily iterate over each element of the array.
We can do in two ways by using "FOR" and "FOR...OF" loop.
Here's how we can do it.

METHOD-1

let myArray = ["N","A","M","E"] for (let i = 0 ; i < myArray.lenght; i++){
console.log(myArray[i]); };

INSIGHT:

Here in our array we let that our indexing is startting from 0, i.e. "i = 0" . For index less than the total length of the array do console.log for (i) and when it's done do 1+ in current "i"(i=index). So, in this way the array will keep on iterating until the value of "i" is less than the length of the array.

METHOD-2

let myArray = ["N","A","M","E"];
for (let char of myArray); console.log(char);

INSIGHT:

What's happening here is , we are defining value every index as "char" of myArray. The 'FOR' then will try to iterate all the char. and console.log it one by one.
