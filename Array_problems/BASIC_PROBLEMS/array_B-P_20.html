Taking out a certain part of an array.

let myArray1 = ["N","A","M","E"];
let myArray2 = ["L","U","V"];
let newArray = MyArray1.concat(myArray2);
console.log(newArray);

RESULT :- ["N","A","M","E","L","U","V"]

INSIGHT:

To add two array into a single array we use the property "CONCAT()". This adds the array in the brackets after adding the first array into the new array which is going to be formed..
